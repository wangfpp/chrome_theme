### 开发一个自己的Chrome主题工具

### 使用方法

- clone此项目到本地

-  Chrome设置 加载已解压的扩展程序


### 功能列表

- 当前日期和天气显示

- 自主切换背景

- 设置搜索引擎


### 开源/免费API

获取geo定位
http://geolocation-db.com

获取天气预警
http://data.cma.cn/kbweb/home/live
